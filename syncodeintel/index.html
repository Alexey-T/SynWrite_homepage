SynCodeIntel plugin

SynCodeIntel is auto-completion plugin for SynWrite text editor. It's based on Code Intelligence library from Komodo Edit text editor.

Version: 0.9.1 (dec 2013)
Download (SynCodeIntel.rar, 13M)

How to install: just open "SynCodeIntel.rar" archive in SynWrite, and confirm installation prompt.

After you install it, it will provide the following functionality for several lexers: PHP, Python, Perl, Ruby, JavaScript, SCSS, LESS, Tcl.

Notes:

  • Code Intelligence library officially supports the following languages (SynWrite may have single lexer for several languages at once, or may not have one): PHP, Python, Python 3, Ruby, Perl, JavaScript, HTML, HTML5, CSS, Mason, XBL, XUL, RHTML, SCSS, XML, XSLT, Django, Twig, Less, Smarty, Node.js, Tcl, TemplateToolkit.
  • It's not recommended for Python, because SynJedi plugin does its Python work better.


1 - Intelligent auto-completion

Put caret after any object name with arrow "->" (for PHP lexer) or with dot "." (for Python and other lexers), and press Ctrl+Space. You'll see intelligent completions list, which is based on type of the object. Partially typed names (after dot) are supported, they just limit completions list.

2 - Function hints

If you type function name and then opening bracket "(", you may call function hint by pressing Ctrl+Shift+Space (usually you don't need to press Ctrl+Shift+Space, as hint is activated automatically, after typing bracket char).


3 - Find symbol definition

If you right-click any object name in source code (class name, function name, variable, etc), and call context menu item "Find ID" (menu item should be enabled after you install SynCodeIntel), plugin will find declaration position of clicked id, and open it for you (in current tab for the same file, or in another tab for another file).

Also you can Ctrl+Alt+click id, instead of using context menu.
