ATPrintPreview component allows to quickly implement "Print preview" dialog for image viewing applications. You can call preview form for a TBitmap/TImage with a single function call.


Preview form

On this form user can change some options and start the actual printing. The preview page on the right is adapted to match the active printer's page sizes.

Form implements the following options:

  • Fit mode
  • Margins
  • Footer text
  • Image position
  • Gamma correction
