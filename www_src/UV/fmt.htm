List of supported formats

  • Documents/Spreadsheets
  • Internet
  • Images
  • RAW Images
  • Audio/Video (separate page)

Documents/Spreadsheets

  • DOC, DOT - Microsoft Word
  • DOCX, DOCM, DOTX, DOTM - Microsoft Word 2007/2010
  • XLS, XLT - Microsoft Excel
  • XLSX, XLTX - Microsoft Excel 2007/2010
  • PDF - Adobe Portable Document Format
  • RTF - Rich Text Format
  • WBK - Microsoft Word backup
  • TXT, NFO, DIZ - Plain text files
  • XPS - Microsoft XML Paper Specification
  • DJV, DJVU - DejaVu document
  • CBR, CBZ - Comic Book archive
  • CHM - Microsoft HTML Help
  • MOBI - Mobipocket e-book
  • EPUB - ePub e-book
  • FB2, FB2Z, FBZ - FictionBook e-book
  • TCR - TCR e-book

    Internet formats

  • ASP, ASPX - Active Server Page script
  • CSS - Cascaded stylesheet
  • HTM, HTML, SHTML, SHTM, STM - HTML page
  • MHT - Microsoft HTML archive
  • PHP - PHP source code
  • XML - XML container
  • XSL - XML stylesheet

    Images

  • ani - Windows animated cursor
  • bmp - Windows bitmap
  • bw - SGI image
  • cel - Autodesk image
  • cur - Windows cursor
  • cut - Dr. Halo image
  • dib - Windows bitmap
  • emf - Windows enhanced metafile
  • fax - GFI fax image
  • gif - Compuserve GIF image
  • icb - Truevision image
  • icl - Windows icon library
  • ico - Windows icon
  • jp2, jpc - JPEG 2000 format
  • jpg, jpeg, jpe, jfif - JPEG format
  • pal - Dr. Halo image
  • pbm - Portable pixmap
  • pcc, pcx - ZSoft Paintbrush image
  • pcd - Kodak Photo-CD image
  • pdd - Photoshop image
  • pgm - Portable pixmap
  • pic - Autodesk image
  • png - Portable Network Graphic
  • pnm - Portable pixmap
  • ppm - Portable pixmap
  • psd - Photoshop image
  • psp - Paintshop Pro image
  • ras - SunOS Unix raster format
  • rgb, rgba - SGI image
  • rla - SGI Wavefront image
  • rle - Windows bitmap
  • rpf - SGI Wavefront image
  • sgi - SGI image
  • tga - TrueVision Targa
  • tif, tiff - Tagged Image File Format
  • vda, vst, win - Truevision image
  • wmf - Windows metafile
