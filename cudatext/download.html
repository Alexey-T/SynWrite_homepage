Note for Windows: For last builds, Python engine is included. For old builds, download Python engine, with the same bitness x32/x64 as CudaText, at SourceForge and unpack zip file to CudaText folder.
Note for Linux and macOS versions: Python engine is not shipped with CudaText. To use plugins on macOS, you must install Python 3.x from official python.org. To use plugins on Linux, see instruction inside "default.json" config about "pylib".
Linux-ARM version is for Raspberry Pi 3, board has Python engine, you must set option "pylib" there.
