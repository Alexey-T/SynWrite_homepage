3rd-party page has some alternative packages.

Note for Linux and macOS versions: Python engine is not shipped with CudaText. To use plugins on macOS, you must install Python 3.x from official python.org. To use plugins on Linux, see instruction inside "default.json" config about "pylib".

Linux-ARM version is for Raspberry Pi 3, see example screenshot how to write option "pylib" there.
